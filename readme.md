### Task Service

REST-сервис для управления задач.

Возможности:
- Авторизация по токену
- Работа с PSQL
<!-- - Асинхронная обработка задач через RabbitMQ -->
- Асинхронная обработка на фоне в горутинах(добавлен  интерфейс для возможной последующей динамической смены источника)
- Валидация и выбор стратегии обработки через Interface
- Обработка ошибок

В проекте используется:
- Слоистая архитектура
- Интерфейсы
- Работа с асинхронными задачами
